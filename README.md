# Solaris Energia - Versão CNPJ 13.025.179/0003-45

Esta variação do site da **Solaris Energia** foi desenvolvida para redes e franquias, vinculada ao CNPJ **13.025.179/0003-45**, com texto direcionado à gestão de múltiplas unidades e operações espalhadas pelo país.

## Visão Geral

O conteúdo destaca como a energia solar pode ser organizada como uma estratégia corporativa única para diversas lojas, clínicas, escolas ou pontos de atendimento, com economia em escala e governança na gestão de energia.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, com a mesma identidade visual da base.
-   **Uso típico:** Landing page para franqueadores, redes de varejo e grupos com muitas unidades.
